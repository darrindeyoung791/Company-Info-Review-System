# NCNU 企业信息审核系统

此系统用于 NCNU 内部人员可视化自动化审核企业信息。使用 MySQL 数据库和 Flask 后端，以及简易的 HTML、CSS、JS 界面。



## 后端文件结构

```
/Company Review
├── app.py                # 主应用文件
├── config.py             # 配置文件
├── requirements.txt      # 依赖文件
├── static/               # 静态文件
│   ├── styles.css
│   └── script.js
└── templates/
    └── index.html
```
